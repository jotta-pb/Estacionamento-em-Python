Sistema de Gestão de Estacionamento
 - Este é um projeto de programação em Python que simula o funcionamento de um estacionamento, desde a entrada do veículo até o módulo de pagamento. O foco do projeto foi praticar lógica de programação, manipulação de strings, estruturas de repetição e organização modular.

· Funcionalidades
  - **Mapa de Vagas Dinâmico:** Exibe um mapa visual (A a J, 1 a 10) no terminal.
  - **Interface Colorida:** Utiliza códigos ANSI para indicar o status das vagas:
     **Verde:** Vaga disponível.
    **Vermelho:** Vaga ocupada.
  - **Módulo de Entrada:** Registro de placa e seleção de vaga com persistência de dados durante a execução.
  - **Módulo de Pagamento (Em desenvolvimento):** Cálcula quantoo motorista irá pagar baseado em variáveis de tempo e preço.
  - **Modularização:** Divisão do código em partes (Entrada e Saída) para melhor manutenção.

· Tecnologias Utilizadas
  - **Linguagem:** Python 3
  - **Interface:** Terminal com códigos de escape ANSI para cores.
  - **Versionamento:** Git e GitHub.
