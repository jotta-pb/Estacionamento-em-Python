Sistema de Gestão de Estacionamento
 - Este é um projeto de programação em Python que simula o funcionamento de um estacionamento, desde a entrada do veículo até o módulo de pagamento. O foco do projeto foi praticar lógica de programação, manipulação de strings, estruturas de repetição e organização modular.

· Tecnologias Utilizadas
  - **Linguagem:** Python 3
  - **Interface:** Terminal com códigos de escape ANSI para cores.
  - **Versionamento:** Git e GitHub.
